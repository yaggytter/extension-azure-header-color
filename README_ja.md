# Header Color Changer for Microsoft Azure

この拡張機能は、複数の Azure AD テナントを扱ったり、複数のユーザを扱う人のために、現在操作している対象がどれかを明確にするためのものです。現在利用中のテナント名やユーザ名(UPN)によって、コンソールのヘッダの色を変更することにより、操作対象を間違えないようにします。
